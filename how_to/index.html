How‑To

เริ่มดูหัวข้อ

วิธีใช้ /spawn

⌨️ Command

หากอยากกลับไปที่สปาวน์ ให้พิมพ์ /spawn ลงในช่องแชทในเกมเพื่อกลับ Spawn

วิธีใช้ /tpa

⌨️ Command

อยากวาร์ปไปหาเพื่อน? พิมพ์ /tpa <ชื่อผู้เล่น> เพื่อส่งคำขอให้เขา จากนั้นให้เพื่อนพิมพ์ /tpac เพื่อยอมรับ หรือ /tpdeny เพื่อปฏิเสธ (หากปล่อยทิ้งไว้นาน คำขอจะหมดอายุ)

วิธีใช้ /back หรือ /bc

⌨️ Command

หากคุณตายและต้องการกลับไปยังจุดที่ตาย ให้พิมพ์ /back หรือ /bc ในช่องแชท ระบบจะพาคุณกลับไปยังตำแหน่งล่าสุดที่ตาย
